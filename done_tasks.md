# Выполненная работа

## Датасет (ЛР №1)

- **Источник:** Сайт Otzovik.com (отзывы)
- **Тип данных:** Текстовые отзывы
- **Количество классов:** 5 (оценки 1–5)
- **Структура:** Папки `dataset/1/` … `dataset/5/`
- **Формат файлов:** `.txt` с ведущими нулями (0001.txt, 0002.txt, …)
- **Метаданные в файлах:** Заголовок, автор, дата, рейтинг, текст отзыва

## Apache Kafka

### Топик
- Название: `raw-data`
- Параметры: `partitions=1`, `replication-factor=1`

### Batch Producer
- **Файл:** `BD_Project/Kafka/batch_producer.py`
- Функции:
  - Чтение отзывов из папок `dataset/<rating>/`
  - Парсинг текстовых файлов
  - Отправка каждого отзыва как JSON в топик `raw-data`
  - Поле `source="batch"`
  - Логирование в `logs/`
  - Поддержка `--limit N` для тестирования

### Consumer
- **Файл:** `BD_Project/Kafka/consumer_hdfs.py`
- Функции:
  - Чтение из топика `raw-data`
  - Сохранение в `temp_hdfs/source=batch/rating=N/date=YYYY-MM-DD/review_*.json`
  - Корректная остановка по `Ctrl+C`
  - Логирование и подсчёт сообщений

## Документация

- **BD_Project/README.md:** Архитектура, структура, инструкции по запуску
- **BD_Project/requirements.txt:** `kafka-python>=2.0.2`
- **BD_Project/.gitignore:** Настроен для исключения датасета и временных файлов

## Окружение

- **Kafka:** Запущен в Docker
- **Хранение данных:** Локальная папка `temp_hdfs/` (для последующего переноса в HDFS)
